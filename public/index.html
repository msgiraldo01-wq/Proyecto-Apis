FLUJO COMPLETO DE DATOS

  1. Navegador  →  GET http://localhost:3000/api/cotizaciones?pais=chile
  2. Express    →  GET https://cl.dolarapi.com/v1/cotizaciones
  3. DolarAPI   →  responde con JSON
  4. Express    →  INSERT INTO cotizaciones (...)  ← guarda en MySQL
  5. Express    →  devuelve el mismo JSON al navegador

ENDPOINTS DEL BACKEND
  GET /api/paises                          → lista de países
  GET /api/cotizaciones?pais=chile         → todas las divisas de Chile
  GET /api/cotizaciones/dolar?pais=chile   → dólar de Chile
  GET /api/historial                       → historial BD
  GET /api/ultimas                         → última por moneda
  GET /api/errores                         → log errores BD
  GET /api/stats                           → estadísticas BD
